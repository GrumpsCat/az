Describe directory services in Azure

What directory services are available in Azure, and what roles do Microsoft Entra ID and Microsoft Entra Domain Services play?

  • Microsoft Entra ID is Azure’s cloud-based identity and access management service that manages users, groups, and applications, and supports authentication, authorization, and single sign-on.
  • Microsoft Entra Domain Services provides managed domain services such as domain join, group policy, and LDAP authentication without requiring customers to deploy and manage domain controllers.
  • Microsoft Entra ID is designed for modern cloud applications and identity federation, while Microsoft Entra Domain Services supports legacy applications that rely on traditional Active Directory features.
  • Both services help organizations centrally manage identities and control access to Azure resources and applications.

Learn more: Microsoft Learn – Directory services in Azure
